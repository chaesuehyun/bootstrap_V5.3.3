폼태그

다양한 폼 사용을 위한 폼 컨트롤 스타일, 레이아웃 옵션, 사용자 정의 검포넌트의 예시와 사용가이드 입니다.

폼테그 사용시 form, fieldset , legend태그 사용

폼 태그 그룹 제목

폼 입력양식의 크기조정은 .form-control-lg, .form-control-sm으로 크기를 조절가능



.form-text을 사용하면 block요소와 간격을 쉽게 맞출 수 있도록 상단 여백이 추가. small, span를 사용하면 좋음 .

패스워드는 특수문자를 포함 8~10까지 가능



버튼 비활성화 - disabled

disabled 불리언 속성을 추가하면 비활성화 외형이 되며, 포인터 이벤트가 삭제되면서 포커싱되지 않습니다



읽기 전용 - readonly

readonly 불리언 속성을 추가함으로써 입력 값의 변경을 막을 수 있습니다. disabled와 달리 readonly입력값은 계속 포커싱 및 선택이 가능합니다.



라디오 버튼 - label태그에 .form-check-label, input태그에 .form-check-input을 적용



체크 박스



라디오 버튼과 같이 label에 .form-check-label, input태그에 .form-check-input 서식을 적용



선택목록 - select, option

select 태그에 .form-select 서식을 적용하고, 크기조정시는 .form-select-lg or .form-select-sm 서식을 적용함

select 태그에 multiple속성을 넣으면 다중선택도 가능. size 속성으로 크기도 조정 가능

다중 텍스트 박스

.formcontrol일때는 가로 100%, form-control-lg form-control-sm을 적용할 수 있음.
